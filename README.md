# Resource Allocator v1

Resource Allocator v1 is a lightweight Python-based system designed to manage and track resource distribution over time.
It focuses on clarity, accuracy, and structured record-keeping, making it suitable for small-scale tracking, analysis, and future extension.

The project emphasizes clean logic, simple data flow, and maintainable code.


# Features

Track allocated resources and current balances

Record inflows and outflows with timestamps

Maintain a clear transaction history

Simple and readable Python implementation

Modular structure for easy updates and scaling

Data-focused logic suitable for analysis or reporting

Beginner-friendly and easy to understand
